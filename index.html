CSS Cheatsheet

Color

Name Purpose
color to set the color of the text.
background-color to set the background color

Text

Name Purpose
font-family allows you to specify the typeface that should be used
font-size allows the font size to be specified
font-weight allows you to create bold text
font-style allows to style your text into italic or oblique
text-transform used to change the case of the text.
text-decoration allow you to decoarate the text with one of the following values:
  • none
  • underline
  • overline
  • line-through
  • blink
line-height sets the height of the entire line of text
letter-spacing controls the spacing between each letter
word-spacing controls the spacing between each word
text-align used to control the alignment of text.
vertical-align To vertically align an inline-level element's box inside its containing line box
text-indent allows the first line of text to be indented.
text-shadow used to create a drop shadow on text.
